Nota

Indicadores de logro
Construye una aplicación web con la implementación de API GraphQL para consultar 
información de forma declarativa.
Introducción a la actividad
En esta actividad, se deberá identificar, instalar y configurar la librería GraphQL dentro de un 
framework de desarrollo web seleccionado (SpringBoot, Express.js, Django, entre otros). El 
objetivo principal es comprender cómo se integra GraphQL en un entorno, reconociendo las 
dependencias necesarias y los pasos de instalación.

3
Descripción de la actividad
1. Seleccionar un framework de desarrollo web (por ejemplo: SpringBoot, Express.js, 
Django, Laravel, etc.) que permita la integración de GraphQL.
2. Identifique la librería adecuada para integrar GraphQL en ese entorno (por ejemplo: 
graphql, apollo-server, graphql-js, graphene, etc.).
3. Utilice el gestor de paquetes correspondiente (maven, npm, pip, composer, etc.) para 
instalar la librería GraphQL y sus dependencias.
4. Grabar un video explicativo donde se muestre el framework, las herramientas utilizadas y 
el proceso de instalación y configuración de GraphQL.
5. La duración del video debe estar entre 3 y 5 minutos. En formato mp4 o enlace a 
plataforma (Y ouTube, Vimeo, con acceso público).
Participación
Para completar esta actividad, cada estudiante deberá participar en el foro siguiendo las 
siguientes acciones:
1. Subir el video explicativo directamente al foro o compartir el enlace de acceso.
2. Acompañar el video o enlace con una breve descripción escrita (máximo 100 palabras) 
que incluya: el framework utilizado y la librería de GraphQL integrada.

4
3. Revisar y comentar el trabajo de dos compañeros, incluyendo una recomendación o 
sugerencia basada en su experiencia, en el material de clase o en referencias externas. 
También, puede comentar sobre el proceso o la configuración realizada.
4. Cada participación debe tener un máximo de 150 palabras. Use un lenguaje claro, 
técnico y respetuoso, enfocado en generar un aporte académico constructivo. Evitar 
comentarios genéricos (“muy bueno”, “me gustó mucho”), sin una base técnica o 
argumentativa.
